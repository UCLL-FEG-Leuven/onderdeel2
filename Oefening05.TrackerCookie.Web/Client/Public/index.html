Tracker cookie

Wanneer je een eerste keer naar deze website surft wordt er een tracker cookie aangemaakt en met de HTTP response als Set-Cookie header teruggestuurd.

Elke keer wanneer je terugkomt naar deze site wordt de cookie terug meegestuurd...

Het aanmaken van de cookie gebeurt door de server: de app.js maakt gebruik van een eigen middleware die gebruik maakt van een tweetal libraries (cookie-parser en uuid). Deze libraries werden ook toegevoegd aan package.json

Als je Chrome gebruikt: cookies kan je terugvinden in de 'Application' tab bij Developer tools.
