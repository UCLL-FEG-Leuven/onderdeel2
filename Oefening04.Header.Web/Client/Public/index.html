Een 15-seconden pagina...

Deze pagina blijft 15 seconden in de cache zitten. Als je binnen die periode de pagina opnieuw zou ophalen zal deze dus niet opgehaald worden bij de server, maar zal de browser de gecachete versie teruggeven.

De Cache-Control header werd toegevoegd door de server.

Indien je test in Chrome, zie SO voor een waarschuwing. Best is dat je gewoon test in een nieuwe tab.

Je kan zien of de pagina uit de cache komt door naar de Network tab te kijken bij Developer tools: bij index.html zou (memory cache) of (disk cache) moeten staan als je binnen de 15 seconden de pagina opvraagt (in een andere tab dus).
